## Getting started

### Installation

- Create project with `create-react-app`
  ```bash
  $ yarn create react-app --template typescript chatfire
  $ cd chatfire
  ```
- Install firebase packages
  ```bash
  $ yarn add firebase react-firebase-hooks
  ```
- Integrate firebase in `src/App.tsx`
